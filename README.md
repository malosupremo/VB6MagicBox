# VB6 Magic Box

Un progetto fenomenale che fa cose a dir poco *strabilianti*!

## Funzioni disponibili (con un pizzico di magia)
1. **Analisi dipendenze**: trova tutti gli intrecci di chi usa chi e dove, senza chiedere il permesso.
2. **Refactoring automatico**: applica naming convention coerenti e rinomina tutto con precisione chirurgica.
3. **Aggiunta tipi mancanti e altro**: completa le dichiarazioni quando il nome suggerisce il tipo giusto 
4. **Riordino variabili locali**: mette ordine in costanti, statiche e variabili, con stile.
5. **Armonizzazione spaziature**: sistema i bianchi secondo le regole del regno (e del `spacingrules.md`).
6. **Bacchetta magica**: fa tutto insieme, con backup e senza panico.
